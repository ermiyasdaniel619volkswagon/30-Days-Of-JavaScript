# 30-Day JavaScript Learning Roadmap (Beginner → Intermediate)

This repository contains a **clear, structured, and practical 30-day JavaScript learning plan**. It is designed to help you move from a **JavaScript beginner** to a **confident intermediate** level by learning core concepts and applying them through hands-on mini projects.

---

## Goal of This Repository

By completing this 30-day roadmap, you will be able to:

- Understand JavaScript fundamentals clearly
- Write clean and readable JavaScript code
- Work confidently with arrays, objects, and functions
- Manipulate the DOM and handle events
- Build small real-world JavaScript projects
- Prepare yourself for learning frameworks like **React.js**

---

## How to Use This Repository

- Follow **one folder per day** (Day 01 → Day 30)
- Read the examples and comments carefully
- Run the code and experiment by changing values
- Practice daily — consistency matters more than speed

⏱ Recommended time: **1–2 hours per day**
